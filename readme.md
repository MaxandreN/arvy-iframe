## Install

1- intall npm pakage : npm i arvy-iframe
https://www.npmjs.com/package/arvy-iframe

2- import 'arvy-iframe' in you js prject
```js
import "arvy-iframe";
```

3- add arvy-card html in you view code
```html
<arvy-card></arvy-card>
```
